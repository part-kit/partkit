# @part-kit/core

Protocol reference implementation for [PartKit](https://partkit.dev) — contract, lockfile, and attestation schemas; part content hashing; the registry client; and the `init` / `add` / `verify` / `guard` operations used by the [`partkit`](https://www.npmjs.com/package/partkit) CLI.

**Status: pre-v0.** Published alongside the CLI's name-claim release; the API is not stable yet. If you are looking for the tool, install [`partkit`](https://www.npmjs.com/package/partkit) instead.
